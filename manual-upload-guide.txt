📋 РУЧНАЯ ЗАГРУЗКА НА GITHUB (без Git)

ПОШАГОВАЯ ИНСТРУКЦИЯ:

1. УСТАНОВИТЕ GIT:
   - Скачайте с https://git-scm.com/download/win
   - Установите с настройками по умолчанию
   - Перезапустите командную строку

2. НАСТРОЙКА GIT (выполните один раз):
   git config --global user.email "ваш-email@example.com"
   git config --global user.name "Ваше Имя"

3. ИСПОЛЬЗУЙТЕ СКРИПТ:
   - Дважды кликните setup-github.bat
   - Или вручную введите команды из файла

4. АЛЬТЕРНАТИВА - Загрузка через веб:
   - Зайдите на https://github.com/MilenaAkulova24/cute
   - Нажмите "Upload files"
   - Перетащите все файлы из папки web-dev-cute-site
   - Нажмите "Commit changes"

5. ПРОВЕРКА:
   - Ваш сайт будет доступен по адресу:
   https://milenaakulova24.github.io/cute/

ВАЖНО: Убедитесь, что в GitHub Pages включен в настройках репозитория (Settings > Pages > Source > Deploy from a branch > main branch)
